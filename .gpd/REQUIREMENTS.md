# Requirements

## Core Requirements

- The state equation, control input operator, target morphology, and control cost must all be explicit.
- The linearized case must produce an exact or numerically stable minimum-energy steering solution.
- Nonlinear simulations must report whether steering reaches the target state or merely approaches it transiently.
- Noise and disorder studies must preserve explicit parameter metadata.
- All claims must be labeled as exact derivation, local result, asymptotic result, numerical result, or interpretive claim.
