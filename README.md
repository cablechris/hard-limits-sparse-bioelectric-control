# Minimum-Energy Bioelectric Control

This repository is a fresh GPD project for the control problem built on the adaptive gap-junction foundations developed in the parent `get-physics-done` workspace.

Core question:

What is the smallest spatiotemporal bioelectric intervention that reliably steers a damaged tissue state into the basin of a target morphology under nonlinear gap-junction dynamics, noise, and disorder?

This repo is intentionally separate from the parent project. The parent repo established the Lyapunov and correspondence baselines. This repo turns that foundation into an optimal-control and steering problem.
